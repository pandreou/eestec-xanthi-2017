ΤΙ ΠΡΕΠΕΙ ΝΑ ΓΝΩΡΙΖΕΤΕ

  • Την δυνατότητα να συμμετέχει σε κάποιο διεθνές event την έχει κάθε μέλος της Τοπικής Επιτροπής της Ξάνθης. Μέλος μπορεί να είναι οποιοσδήποτε φοιτητής,προπτυχιακός και μεταπτυχιακός, του τμήματος Ηλεκτρολόγων Μηχανικών και Μηχανικών του ΔΠΘ.

  • Η συμμετοχή στα περισσότερα event είναι δωρεάν, με εξαίρεση κάποια τα οποία σας ενημερώνουν για κάποιο μικρό fee πριν την αίτηση σας για συμμετοχή. Σε κάθε περίπτωση, καλύπτονται πλήρως τα έξοδα σας για διατροφή και διαμονή.

  • Μπορείτε να ενημερώνεστε για τα τρέχοντα event με εξης τρόπους:

    - Ενημερώνουμε τα μέλη μας μέσω mailing list.
    - Ανεβάζουμε ενημερώσεις στο ανοιχτό group μας στο Facebook, EESTEC Xanthi
    - Μπορείτε να ενημερώνεστε κατευθείαν απο την κεντρική σελίδα της EESTEC .
